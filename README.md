# Fire and Smoke Detection System (IoT)

This project is an IoT-based system designed to detect smoke and fire hazards and provide alerts.

## Features

* Real-time smoke and fire detection
* Sensor-based monitoring system
* Alert mechanism for safety

## Technologies Used

* Arduino
* Smoke and Fire Sensors

## Description

The system detects hazardous conditions using sensors and triggers alerts to ensure safety. It demonstrates practical implementation of IoT in real-world scenarios.
